Internet Explorer 4.01
for Macintosh

Version 4.01 · 1998
1998 · Mac OS 9 (Platinum)

For five years, the default browser on every Mac was made by Microsoft — and it was surprisingly good.

A short history

In 1997, a near-bankrupt Apple struck a dramatic truce with Microsoft. Part of the deal: Internet Explorer became the default browser on the Mac. At Macworld Boston, Steve Jobs announced a $150 million Microsoft investment while Bill Gates loomed on a giant screen overhead — to a chorus of boos from the crowd.

The Mac team built their own browser, sharing the IE name and the blue "e" but almost none of the Windows code. It ran in the classic Mac OS "Platinum" look and, with the later IE 5 for Mac, earned a real reputation for quality.

Lesser-known & curious

  • It out-standards-ed Windows IE. Its "Tasman" engine had some of the best CSS support of its time — cleaner than Microsoft's own flagship browser.
  • Two browsers, one logo. Mac IE and Windows IE shared a name and an icon but were almost entirely separate products.
  • It pioneered Mac-friendly touches like a Scrapbook, a "Page Holder" side panel, and a genuinely good Print Preview.
  • The spinning blue "e" served as its loading throbber, framed in a little black box on the toolbar.
  • Apple ended it. When Safari shipped in 2003, the deal lapsed and Microsoft discontinued Mac IE that same year.
  • Its Tasman engine quietly influenced Microsoft's later layout work for years afterward.
