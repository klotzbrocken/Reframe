Netscape 7.02
for Macintosh

“Modern” skin · February 2003
Mozilla-based · Mac OS 9 “Platinum” chrome

The last great Netscape — Mozilla under the hood, wrapped in the brushed blue-grey metal and big round buttons of the “Modern” theme.

About this recreation

Built from Netscape 7.02's original shipped skin. Because the browser was Mozilla-based, its interface was plain CSS + images — so the toolbar buttons, the Location bar's multi-colour bevel and every colour value here are taken verbatim from the stylesheets Netscape actually released, not reconstructed by eye. The page itself is the live modern web.

Period details

  • Brushed-metal toolbar — the signature #C7D0D9 blue-grey band with a soft highlight along the top edge.
  • Round metal buttons — Back, Forward, Reload and Stop are the original 41×38 buttons, each with its own normal / hover / pressed / disabled artwork.
  • The Netscape throbber — sits still until a page loads, then spins.
  • “Location:” bar — a three-pixel multi-colour bevel with a black inner line over the #EDF5F7 field tile.
  • Personal Toolbar — the bookmark quick-links bar below the address.
