Internet Explorer 5

Version 5.0 · 1999
March 1999 · Windows 98

The browser that won the first browser war — and quietly invented the technology that powers the modern web.

A short history

Released in March 1999 and bundled with Windows, Internet Explorer 5 (with 5.5 and 6 close behind) pushed Microsoft's share past 90%. It brought dynamic HTML, editable pages, bi-directional text and "Favorites" with offline reading. For most people of the era, the blue "e" simply was the internet.

That dominance came at a price: bundling the browser with Windows to crush Netscape sat at the very center of the landmark antitrust case United States v. Microsoft, which nearly saw the company broken in two.

Lesser-known & curious

  • It invented AJAX. IE5 introduced XMLHttpRequest — the object behind virtually every modern web app. Microsoft built it for Outlook Web Access; the rest of the world adopted it years later.
  • The Mac version was better at standards. IE5 for Mac used a completely different engine ("Tasman") with far stronger CSS support than the Windows edition.
  • AutoComplete for web addresses and form fields debuted here — and never left.
  • It shipped with a built-in "Radio" toolbar that streamed internet radio stations.
  • Its quirky, incomplete CSS — especially the "box model" — haunted web developers for the next decade.
  • So many people clicked the "e" that a generation grew up believing the letter was the web itself.
