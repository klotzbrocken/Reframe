K-Meleon 0.2

Version 0.2 · 2001
2001 · Windows 98

A featherweight browser that paired Netscape's powerful new engine with a plain, fast, native Windows skin.

A short history

First released in 2000, K-Meleon used Mozilla's Gecko engine — the same one inside Netscape 6 — but wrapped it in a lightweight native Windows interface instead of the heavier cross-platform toolkit. The result rendered the modern web while staying tiny and quick, even on older PCs that struggled with Netscape and Internet Explorer.

It never chased the mainstream. Instead it became a quiet favorite of people who wanted Gecko's power without the weight.

Lesser-known & curious

  • The name is a pun. "K-Meleon" plays on chameleon — a wink at Gecko, the engine it rides on.
  • Native, not XUL. By using real Windows controls instead of Mozilla's cross-platform UI, it launched faster and used far less memory.
  • It was endlessly customizable through macros and config files — a true power-user's browser.
  • It was deliberately Windows-only, trading portability for raw speed.
  • It earned a devoted niche on low-spec and aging machines, and a small community kept it alive for many years.
  • It quietly proved a big idea: a rendering engine and its interface can be completely separate — something browsers still rely on today.
