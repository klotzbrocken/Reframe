NCSA Mosaic

1993 · NCSA, University of Illinois
The browser that showed the world the Web

Not the first web browser — but the one that made the World Wide Web something ordinary people wanted to use.

A short history

Mosaic was built from 1992 by a small team at the National Center for Supercomputing Applications (NCSA) at the University of Illinois at Urbana-Champaign, led by Marc Andreessen and Eric Bina. Version 1.0 shipped in 1993 for Unix/X, then Windows and Macintosh — the first browser most people ever saw, and the first to run on all three platforms.

Its key move was showing images inline with the text (via the <img> tag) instead of in separate windows, and wrapping it all in a friendly point-and-click interface. Within a year the Web went from a physics-lab tool to a mainstream phenomenon.

Lesser-known & curious

  • There is no address bar. You opened a page through File ▸ Open URL…; the “Document Title” and “Document URL” boxes at the top are read-only displays — recreated here faithfully.
  • The spinning globe. The NCSA globe with orbiting network arcs (top right) was the original “throbber” — it animated while a page loaded, an idea every browser copied.
  • Grey pages, blue links. Before Netscape, pages had a #C0C0C0 grey background and fixed-width text — white backgrounds came later.
  • It spawned Netscape. Andreessen left to co-found Netscape, whose Navigator (and its “Mozilla” — “Mosaic killer”) overtook Mosaic and lit the browser wars.
  • Microsoft licensed it. Internet Explorer was built on Mosaic code licensed from Spyglass — so the blue “e” has Mosaic in its DNA.
  • NCSA retired Mosaic in January 1997, the era this theme is cut from.
