Internet Explorer 1.0

Version 1.0 · 1995
August 1995 · Windows 95

The first shot of the browser wars almost nobody noticed — a bare-bones browser you had to pay extra for.

A short history

Internet Explorer 1.0 shipped in August 1995 as part of the Microsoft Plus! add-on pack for Windows 95 — a paid extra, not yet built into the operating system. Under the hood it wasn't really Microsoft's work: the code was licensed from a company called Spyglass, whose browser was itself derived from NCSA Mosaic, the program that first put the graphical web on ordinary screens.

It was about as simple as a browser can be: basic HTML and hyperlinks, no JavaScript, no stylesheets, no frames. Arriving roughly three months after Netscape Navigator 1.0, it made almost no dent in Netscape's commanding lead — but it was the opening move in a fight that would reshape the entire industry.

Lesser-known & curious

  • It was Mosaic in disguise. The About box quietly credited the original: "Based on NCSA Mosaic… developed at the National Center for Supercomputing Applications."
  • The licensing deal backfired spectacularly. Spyglass was promised a royalty on every copy sold — so Microsoft gave IE away for free, paid almost nothing, and Spyglass sued. The fight was settled for around $8 million.
  • Built in a hurry. A small team turned the licensed code into a shipping product in a matter of weeks.
  • Barely a browser by today's standards. No tabs, no history sidebar, no real support for the fancier pages of the day.
  • Version 1.5, for Windows NT, added something as basic as rendering HTML tables.
  • The famous globe-and-"e" logo didn't exist yet — branding was an afterthought.
